Java踩坑记录

Java 小小总结

🌳错误：找不到符号

使用通配符进行批量编译

🌳jar中没有主清单属性

  • 用压缩包的方式打开jar文件
  • 打开META-INF文件夹,看到MANIFEST.MF文件
  • 添加一句代码 Main-Class: xxxxx

🌳找不到或无法加载主类

  1. 编译时需要加-d . (有package)
  2. 运行时需要加包名

🌳xx不是内部或外部命令

  1. 重启cmd
  2. 未配置环境变量，或将%%中间内容替换为实际路径

🌳类中找不到符号

记得import组件文件

🌳莫名报语法错误

可能是自动import了不需要的包
